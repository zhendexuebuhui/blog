Linux zsh终端安装配置

Tim   in Shanghai
2022-08-30
  • zsh
  • 服务器运维

# 1 安装

Dev ：Ubuntu 16.04

sudo apt-get install curl
sudo apt-get install git

# 安装zsh
sudo apt-get install zsh
# 安装on my zsh
sh -c "$(curl -fsSL https://gitee.com/shmhlsy/oh-my-zsh-install.sh/raw/master/install.sh)"

重启终端后

vi ~/.bashrc
# 添加
if [ -t 1 ]; then
    exec zsh
fi

重启终端后即，默认的shell就变成了zsh

# 2 配置

和bash不同，zsh的配置文件是~/.zshrc，实际上oh-my-zsh的默认配置也够我们使用了，但是这样其真正的强大之处并不能得到很好的体现，因此我们可以继续看看对应的插件和主题功能。

# 2.1 主题

oh-my-zsh的主题非常丰富，可以用如下命令查看已有主题：

ls .oh-my-zsh/themes

个人比较喜欢简单的，因此用了ys主题，进入.zshrc配置文件进行修改

vim ~/.zshrc

将ZSH_THEME改为ZSH_THEME="ys"，然后:wq保存退出，主题就自动生效

# 2.2 插件

oh-my-zsh的插件生态非常丰富，下面列出来的是我个人比较常用的插件，如果你有兴趣，可以取发掘能提高自身效率的插件~

注意：如果操作过程中出现_arguments:448: _vim_files: function definition file not found错误，请执行：rm -f ~/.zcompdump即可

部分插件地址：

  1. autojump (opens new window)
  2. zsh-autosuggestions (opens new window)
  3. zsh-syntax-highlighting (opens new window)
