# Rust开发环境配置

# Windows本地

# 安装Rust

点击链接下载: https://rustup.rs/

运行下载好的exe文件，命令行输入1按下回车即可。

安装完毕后在CMD输入rustc -V和cargo -V，如果能输出版本号则表示安装成功。

# VSCode安装插件

VSCode中按下Ctrl+Shift+X打开扩展商店，安装以下插件：

  • rust-analyzer
  • Rustfmt（用于格式化代码）
  • Native Debug（用于本地启动调试）
  • Rust （提供工程初始化、导航、格式化等）
  • crates （对cargo.toml进行管理）

# Linux服务器

# 安装通用编译开发环境

包含gcc、gdb、make等

Debian、Ubuntu安装

$ sudo apt update 
$ sudo apt-get install build-essential

CentOS系安装

$ sudo yum groupinstall "Development Tools" "Development Libraries"

# 安装Rust

$ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
## 查看版本号
$ cargo --version
$ rustc --version
